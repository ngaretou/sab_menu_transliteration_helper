Download the file and import to SAB

You should now have a good file you can import back into SAB in Appearance/Interface in the Translations tab. If you have not yet added your transliteration into SAB, just be sure to add the language in Appearance/Interface in the Lanugages tab with the same language code.

If you see problems with variables like %s or %book, you can deal with them in TECkit map by adding something like this after the SFM section: 

----------------------------------------
; ----------

; Scripture App Builder (SAB) related 

; ----------

'%,d'                      >       '%,d'

'%book '                   >       '%book '

'%chapter '                >       '%chapter '

'%s'                       >       '%s'

----------------------------------------


If something goes wrong or takes some time to figure out, you can go back to your earlier exported file.
